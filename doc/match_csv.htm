Installation
  1. Install Java, see the web site https://www.java.com/en/download/ .
  2. Install R, see the web site https://www.r-project.org/ .
  3. Install Git, see the web site https://git-scm.com/downloads .
  4. Download match_csv by executing the following command: 
     
     git clone https://github.com/bradbell/match_csv.git match_csv.git
  5. Open R, use getwd() to determine your current directory. Use setwd('path') to change into the match_csv.git directory where path is the path on your match_csv.git directory.
  6. Execute the command dir.create('build') to create the match_csv.git/build directory.
  7. Execute the command
         source('example/student_college/get_started.R')
     This runs the get_started.R example. After running this example, the build directory will contain the two input files student.csv, college.csv and the output file match.csv
