Installation

Java
Install Java, see the web site https://www.java.com/en/download/ .

R
Install R, see the web site https://www.r-project.org/ .

Download match_csv
If you have or can install git
     https://git-scm.com/downloads
download match_csv by executing the following command: 
 
     git clone https://github.com/bradbell/match_csv.git match_csv.git
In this case the distribution directory is match_csv.git. Otherwise, choose one of the versions on match_csv
     https://github.com/bradbell/match_csv/releases
page and unpack it. In this case the distribution directory is the one created when you unpack it.

Distribution Directory
The directory you created using the download instructions above is referred to as the distribution directory. For example, if you used git, the distribution directory is match_csv.git.

matchingMarkets
Open R and following the install instructions for https://github.com/thiloklein/matchingMarkets .
  1. If you have an error related to the gmp package, make sure the related libraries are installed; see gmp
  2. If you have problems loading rJava; see R: rJava package install failing
