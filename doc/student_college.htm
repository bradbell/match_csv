Student / College Matching Problem

Description
Finds one of the stable matchings for the college admissions problem.

Assumptions
  1. No two students have the same name and no student name is empty.
  2. No two colleges have the same name and no college name is empty.


n_student
We use n_student to denote the number of students in the match, which is also equal to the number of columns in student_file .

n_college
We use n_college to denote the number of colleges in the match, which is also equal to the number of columns in college_file .

student_file
This is a comma separated value file with n_student columns and n_college+1 rows. The first row contains the names of the students. The other entries in each column are the names of the acceptable colleges for the student name in the first row of the column. The lower the row index, the more preferred the college is to the student. No college name appears twice in a column. Empty rows, for the colleges that do not appear in a column, are all at the bottom of the column.

college_file
This is a comma separated value file with n_college columns and n_student+2 rows. The first row contains the names of the colleges. Each column of the second row contains a positive integer that specifies the number of students the corresponding college is willing to accept. The other entries in each column are the names of the acceptable students for the college name in the first row of the column. The lower the row index, the more preferred the college is to the student. No student name appears twice in a column. Empty rows, for the students that do not appear in a column, are all at the bottom of the column.

result
The return value result is a data frame with column names student and college. Each row of the result is a matched pair of a student to a particular college. The number of rows nrow(result) is the number of pairs. The student and college for the i-th pair are result[i,"student"] and [i,"college"] respectively.

Example
The file student_college_s7_c2_a3.R contains an example and test using student_college.
Input File: R/student_college.R
